# Visitor Management System

A full-stack Visitor Management System (VMS) built to automate and manage visitor records efficiently with enhanced security and real-time tracking.

---

## 🚀 Features

- 🔐 Admin Authentication (JWT based login)
- 📝 Visitor Registration System
- 🆔 Auto-generated Visitor ID
- 📸 Photo Capture using Webcam
- ✍️ Digital Signature (E-Signature)
- ⏱️ Check-in and Check-out Tracking
- 📊 Dashboard with Analytics & Charts
- 📁 Secure Data Storage (MongoDB)

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Axios
- Recharts

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB
